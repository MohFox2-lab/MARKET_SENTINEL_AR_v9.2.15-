# دليل الاستخدام السريع (Quick Start)

مرحباً بك في Market Sentinel! للبدء في استخدام النظام، اتبع الخطوات البسيطة التالية:

## الخطوات الأساسية

1. **فك الضغط وفتح التطبيق**: 
   - فك ضغط ملف ZIP
   - افتح ملف `index.html` مباشرة في متصفحك (Chrome, Edge, Firefox)
   - لا تحتاج إلى أي إعدادات إضافية

2. **تجهيز البيانات**: 
   - جهز ملف بيانات السوق (CSV, JSON, أو Excel)
   - يجب أن يحتوي على: التاريخ، سعر الفتح، الأعلى، الأدنى، الإغلاق، حجم التداول
   - يمكنك استخدام العينات المرفقة في مجلد `datasets/`

3. **رفع الملف والتحليل**:
   - اضغط زر "رفع ملف" في الواجهة
   - اختر ملف البيانات الخاص بك
   - النظام سيحلل البيانات تلقائياً

4. **قراءة النتائج**:
   - **التوصية**: شراء / زيادة الكمية / احتفاظ / تصريف جزء / بيع
   - **الأسباب**: عرض مفصل لسبب التوصية (Smart Money, السيولة, VWAP, إلخ)
   - **المؤشرات**: جميع المؤشرات الفنية والتحذيرات

5. **أدوات إضافية**:
   - **تحليل متعدد الأسهم**: اختبر عدة أسهم في نفس الوقت
   - **Portfolio**: راقب أسهمك وحلل المحفظة بالكامل
   - **Screener**: ابحث عن فرص استثمارية

## فهم التوصيات

| التوصية | معنى | Trust Score |
|---------|------|-------------|
| شراء / زيادة الكمية | فرصة شراء قوية جداً | >= 80 |
| شراء | فرصة شراء جيدة | >= 70 |
| احتفاظ / تحت المتابعة | وضع محايد - اترقب | >= 55 |
| تصريف جزء منها | بدء تقليل التعرض | 40-54 |
| بيع | خطر عالي - بيع سريع | < 40 |

## ملاحظات مهمة

- التطبيق **Static** - يعمل محلياً بدون اتصال بسرفر
- النظام مبني على **مؤشرات فنية فقط** بدون ذكاء اصطناعي
- التوصيات **دعم قرار** وليست توصية مالية رسمية
- طبق **إدارة مخاطر صارمة** دائماً
