# توثيق سريع للهيكل

## المجلدات المهمة
- engines/ : محركات التحليل (كل محرك مستقل).
- js/ :
  - app.js: التشغيل + الربط + التنقل بين التبويبات
  - analysisEngine.js: تجهيز البيانات واستدعاء المحركات
  - advancedUI.js: رسم الواجهات (Cards/Views) بناءً على نتائج التحليل

## مبدأ العمل
1) المستخدم يطلب تحليل رمز.
2) analysisEngine يبني analysis object.
3) يتم استدعاء المحركات داخل engines/.
4) advancedUI يعرض النتائج بواجهة عربية تفسيرية.

## ملاحظة الأداء
لتحسين الأداء مستقبلًا بدون تغيير الهيكل:
- نقل الحسابات الثقيلة إلى Web Worker.
- دمج/تصغير ملفات JS في نسخة Production.
